Round 1 Operations

Each byte in the state matrix is replaced with a sub-byte using an 8-bit substitution box, the Rijndael S-box. This operation provides the non-linearity in the cipher.

info
Example: Byte 63 is mapped to FB. This is found by looking at the 6th row and 3rd column of the AES S-Box.
The rows of the state are shifted cyclically to the left. The first row is left unchanged, the second is shifted one position, the third by two, and the fourth by three.
Each column of the state is multiplied by a fixed polynomial modulo x^4 + 1. This step provides diffusion by spreading the influence of each input byte over multiple output bytes.
The state is combined with a round key using bitwise XOR. The round key is derived from the original cipher key using the Rijndael key schedule.
