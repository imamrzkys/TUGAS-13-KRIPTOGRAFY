Each byte in the state matrix is replaced with a corresponding byte from the Rijndael S-Box. This provides non-linearity in the cipher.
lookup(0x63) -> 0xFB
The last three rows of the state are shifted cyclically a certain number of steps. This ensures that the columns are mixed in the long run.
Each column of the state is multiplied by a fixed polynomial. This operation provides diffusion in the cipher.
A round key is combined with the state. For each round, a subkey is derived from the main key using the AES key schedule.
