01 SubBytes Transformation
expand_more
Non-linear substitution step where each byte is replaced with another according to a lookup table (S-Box). This provides confusion in the cipher.
Input: 0x32 → S-Box[0x32] → Output: 0x23
02 ShiftRows Operation
expand_more
A transposition step where the last three rows of the state are shifted cyclically a certain number of steps.
03 MixColumns Layer
expand_more
Linear transformation operating on the columns of the state, combining the four bytes in each column.
04 AddRoundKey
expand_more
Bitwise XOR of the state with a subkey derived from the main key. This is the only step that uses the secret key.
