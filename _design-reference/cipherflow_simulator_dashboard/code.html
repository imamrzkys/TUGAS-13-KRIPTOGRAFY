Laboratory Insight

The initial round adds the subkey to the state array using a bitwise XOR operation. This ensures that every bit of the cipher depends on the key from the very start.

Did you know? In the Initial Round (AddRoundKey), the state is combined with the main key using bitwise XOR. This is the only round that uses the original key directly before it gets transformed by the key schedule.
